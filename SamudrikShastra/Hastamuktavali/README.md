# Hastamuktavali — हस्तमुक्तावली

**Author:** Unknown
**Period:** Medieval (date uncertain)
**Status: Not yet digitized in this collection**

Specialized palmistry text — classifies auspicious and inauspicious marks on the palm including fish (matsya), trident (trishula), conch (shankha), and flag (dhvaja) symbols. Used as a reference for Hasta Samudrika (hand reading) practice.

Available at: Archive.org — https://archive.org/stream/in.ernet.dli.2015.125592
