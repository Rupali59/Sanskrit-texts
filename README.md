# Sanskrit Texts Repository

An open-source collection of classical Sanskrit texts, meticulously digitized and structured for computational parsing and programmatic access. This repository aims to preserve ancient knowledge systems—particularly Jyotisha (Vedic Astrology)—in modern formats like Markdown and JSON.

## Contents
- **BrihatParasharaHoraShastra (BPHS)**: The foundational text of Vedic Astrology, available with original Sanskrit shlokas and programmatic JSON structures including translations.

## Structure
Each text typically contains:
- `*.md`: The source markdown containing Devanagari text.
- `*.json`: Parsed, structured data (chapters, shlokas, translations) suitable for APIs.
