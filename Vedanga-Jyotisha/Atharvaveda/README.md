# Atharvaveda Vedanga Jyotisha

**Status: Lost — no extant manuscript located.**

An Atharvaveda recension of the Vedanga Jyotisha is mentioned in the textual tradition but no manuscript has been found. The Rigveda and Yajurveda recensions (both attributed to Lagadha) are the only surviving versions.

Sources: Vedic Heritage Portal (vedicheritage.gov.in), scholarly surveys of Vedanga literature.
