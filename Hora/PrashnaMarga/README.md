# Prashna Marga — प्रश्नमार्ग

**Author:** Unknown (Kerala Jyotish tradition)
**Period:** 16th–17th century CE
**Status: Not yet digitized in this collection**

The primary text of the Kerala horary (Prashna) tradition. Covers answering questions through chart analysis at the moment of inquiry — distinct from natal astrology. Considered authoritative for Prashna (horary) in the Kerala school.

Available at: Internet Archive, various Sanskrit presses.
