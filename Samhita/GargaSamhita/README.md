# Garga Samhita — गार्गसंहिता

**Author:** Garga (traditional attribution)
**Period:** Ancient — date disputed, possibly pre-Varahamihira
**Status: Not yet digitized in this collection**

Garga is one of the earliest named Jyotish authorities, cited by Varahamihira and others. The Garga Samhita covers omens, planetary portents, and mundane predictions. Multiple recensions exist; the complete text is difficult to establish.

Note: distinct from the devotional *Garga Samhita* (on Krishna's life), which is a Vaishnava Purana.

Available at: Internet Archive (partial), Indian manuscript repositories.
