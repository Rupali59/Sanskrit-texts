# Samhita — संहिता (Encyclopedic / Mundane Astrology)

The Samhita skandha covers collective phenomena: meteorology, agriculture, politics, omens, architecture (Vastu), medicine, and natural events. Where Hora is individual, Samhita is societal.

Varahamihira's Brihat Samhita is the canonical reference — 92 chapters spanning astronomy, astrology, divination, and encyclopedic knowledge of the ancient world.
